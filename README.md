# Gestor de Canchas de Fútbol Sintético

Aplicación móvil desarrollada en React Native con Expo para la gestión integral de canchas de fútbol sintético, incluyendo reservas, finanzas y torneos.

## Definición del problema

Muchos administradores de canchas de fútbol sintético carecen de una herramienta digital que integre la gestión de reservas, el control de mantenimiento, el registro financiero y la organización de torneos. Esta aplicación soluciona esa necesidad ofreciendo una interfaz móvil intuitiva y funcional.

## Requisitos previos

- Node.js (versión 14 o superior)
- npm o yarn
- Expo CLI (instalado globalmente: `npm install -g expo-cli`)
